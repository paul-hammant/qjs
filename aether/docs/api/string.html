string_to_int_raw

string_to_int_raw(str, out_value)

Parsing (string -> number) — raw form with out-parameter. Returns 1 on success, 0 on failure. Result stored in out_value. Aether code should prefer the Go-style wrappers in std.string (`string.to_int`, etc.) which return `(value, error)` tuples.
